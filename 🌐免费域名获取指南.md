## 🔧 如何在 Vercel 中添加自定义域名

如果你购买了域名，在 Vercel 中添加的步骤：

### 步骤 1：在 Vercel 中添加域名

1. 在 Vercel 项目页面，点击 **"Settings"** → **"Domains"**
2. 点击 **"Add Domain"**（添加域名）
3. 输入你的域名（例如：`mymeal.com`）
4. 点击 **"Add"**

### 步骤 2：配置 DNS 记录

Vercel 会显示需要配置的 DNS 记录，通常是：

**方法 A：使用 CNAME 记录（推荐）**
```
类型: CNAME
名称: @ 或 www
值: cname.vercel-dns.com
```

**方法 B：使用 A 记录**
```
类型: A
名称: @
值: 76.76.21.21（Vercel 提供的 IP）
```

### 步骤 3：在域名注册商配置 DNS

1. 登录你的域名注册商（如 Namecheap）
2. 找到域名管理 → DNS 设置
3. 添加 Vercel 提供的 DNS 记录
4. 等待 1-24 小时生效（通常几分钟到几小时）

### 步骤 4：等待 SSL 证书自动配置

- Vercel 会自动为你的域名配置 SSL 证书（HTTPS）
- 通常需要几分钟到几小时
- 配置完成后，你的网站就可以通过自定义域名访问了
